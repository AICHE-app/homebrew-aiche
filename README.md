# AICHE Homebrew Tap

Custom Homebrew tap for AICHE Desktop - AI-powered audio transcription app.

## Installation

```bash
brew tap AICHE-app/aiche
brew install --cask aiche-desktop
```

## What is AICHE Desktop?

AICHE Desktop is an AI-powered audio transcription and enhancement application for macOS. It provides real-time transcription with global hotkeys support.
